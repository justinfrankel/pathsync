PathSync version history:

v0.33 (3/27/06): 
+ fixed a bug when using !dirname\ in expression list (thanks dave biderman)

v0.32 (11/07/05):
+ added greyed "ignore file date" checkbox, which ignores differences
  of exactly 1h and 2h, for daylight savings time nonsense.

v0.31 (4/20/05):
+ fixed bug in filename matching
+ fixed throttling high CPU use

v0.3 (4/20/05):
+ commandline option -systray (courtesy Francis Gastellu)
+ filename pattern matching options (courtesy Francis Gastellu)
+ -autorun command line option (courtesy Alan Davies) 
+ logging option (useful for -autorun or normal use) (courtesy Alan Davies)
+ optional copying throttling (coming soon: throttling for file scanning)


v0.2 (3/08/05):
+ support for PSS (pathsync settings) files 
  + menu items to load/save
  + drag&drop PSS files
  + shell integration (open your PSS in explorer, or run pathsync -loadpss <file>)
+ option for ignoring dates/sizes/missing files
+ option for default sync actions
+ installer

v0.1 (8/20/04):
+ first release
